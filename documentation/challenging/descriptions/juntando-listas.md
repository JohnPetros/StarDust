<Text>Em um futuro distante, várias espécies alienígenas se unem para explorar o espaço. Cada espécie possui uma lista de espaçonaves identificadas por números, letras ou combinações.</Text>

<Text>Para facilitar a união, você deve unir as listas de cada espécie em uma só, para que todos possam explorar juntos.</Text>

<Quote title="Exemplo 3">Entrada: `[1, 3, 5], [2, 6, 8]`, Saída: `[1, 3, 5, 2, 6, 8]`</Quote>

<Quote title="Exemplo 3">Entrada: `[7, 8], [10, 9, 1, 1, 2]`, Saída: `[7, 8, 10, 9, 1, 1, 2]`</Quote>
